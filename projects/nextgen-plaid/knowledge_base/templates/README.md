# Knowledge Base Templates

This directory contains copy/paste templates for creating consistent docs under `knowledge_base/`.

## Templates

- `0000-EPIC-OVERVIEW-template.md`
  - Copy into an epic directory and rename to `0000-overview-...`. Use this for epic overview + PRD summary table.
- `PRD-template.md`
  - Copy into an epic directory and rename to `PRD-<epic>-<nn>-<slug>.md`.
- `0001-IMPLEMENTATION-STATUS-template.md`
  - Copy into an epic directory and rename exactly to `0001-IMPLEMENTATION-STATUS.md`.

## Convention (based on `knowledge_base/epics/wip/NextGen/Epic-4`)

Within an epic directory:

- `0000-overview-...md` — Epic overview and decisions
- `0001-IMPLEMENTATION-STATUS.md` — Rolling implementation tracker (update after each PRD)
- `PRD-<epic>-<nn>-<slug>.md` — Individual PRDs
