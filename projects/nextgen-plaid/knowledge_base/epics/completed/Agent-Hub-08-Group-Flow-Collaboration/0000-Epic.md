# Epic 8: Group Flow & Collaboration (Agent-Hub-08)

### Epic Goal
Enable complex inter-agent collaboration and improved input capabilities, allowing the owner to monitor agent-to-agent interactions and provide rich context through file uploads and @mentions.

### Scope
Workflow Monitor tab (Read-only "back and forth" view); @mentions for persona routing; File uploads (Active Storage) for runs; Dynamic command autocomplete.

### Non-Goals
Core persistence (Epic 7); Advanced UI controls (Epic 6).

### Dependencies
Epic 3 (Input Bar), Epic 5 (Smart Commands).

### Risks/Mitigations
Information overload in Workflow Monitor → "Less is more" focus on key handoffs; File size → Cap uploads.

### End-of-Epic Capabilities
- Users can monitor inter-agent exchanges in a dedicated Workflow Monitor tab.
- @mentions (e.g., @SAP) correctly route messages to the specified persona.
- Files (logs, schemas) can be attached to conversations via the input bar.
- Autocomplete suggests persona-specific commands dynamically.

### Safety Policy
- Workflow Monitor is read-only.
- File uploads scanned/restricted to safe types.
- Personas only access their designated scope.
