# Epic 3: Componentized UI (Agent-Hub-03)

### Epic Goal
Modularize UI with ViewComponents for tabs, sidebar, input—enables maintainable, reusable elements as hub scales, reducing bloat.

### Scope
Persona tabs/switching; collapsible sidebar with badges; input bar with autocomplete/uploads; sidebar search/filter.

### Non-Goals
Persistence/CRUD; smart commands.

### Dependencies
Epic 2 for pane; Stimulus for JS.

### Risks/Mitigations
View bloat → component previews/tests; high volume → search scopes.

### End-of-Epic Capabilities
- Switch personas with color-coded updates.
- View/select convos with pending badges.
- Start convos via input with attachments.
- Search/filter sidebar for scalability.

### Safety Policy (Canonical)
- No destructive actions; read-only views.
- Scopes prevent data leaks.
- Mobile collapse for usability.

### Model Discovery
Not applicable; prep for later overrides.
