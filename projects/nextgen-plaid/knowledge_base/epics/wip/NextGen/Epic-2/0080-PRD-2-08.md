# PRD-2-08: Snapshot Export API

## Overview
Implement API endpoints for exporting snapshots as JSON, including sanitized RAG context for AI ingestion, using DataProvider exports. Enables intern downloads and AI curriculum queries.

## Status
**Branch**: `feature/epic-2-financial-snapshots`
**Dependencies**: PRD-2-07 (Admin Preview)

## Requirements

### Functional
- **Routes**:
  - `/api/snapshots/:id/download` - Full JSON, user-owned only
  - `/api/snapshots/:id/rag_context` - Sanitized JSON for AI

- **Download Endpoint**:
  - Returns full `snapshot.data` as JSON
  - Uses `send_data` with filename: `"financial-snapshot-#{snapshot.snapshot_at.to_date}.json"`
  - Content-Type: `application/json`
  - Auth: User must own snapshot

- **RAG Context Endpoint**:
  - Returns sanitized JSON via `DataProvider#to_rag_context`
  - Strips sensitive fields:
    - `account_numbers`
    - `institution_ids`
    - `raw_transaction_data`
  - Adds:
    - `user_id_hash`: SHA256 of user.id + ENV['RAG_SALT']
    - `exported_at`: Current timestamp
    - `disclaimer`: Enhanced for RAG usage
  - Auth: Admin/AI service only (check for admin or API key)

- **BI Stub**: `DataProvider#to_tableau_json`
  - Returns flattened hash for easy import (e.g., `allocation_equity: 0.6` instead of nested)
  - Stub implementation for v1

### Non-Functional
- Auth: Owner for download, admin/API-key for RAG
- Performance: <1s response time
- Privacy: Sanitization ensures no PII in RAG context

## Acceptance Criteria
1. `/api/snapshots/:id/download` returns full JSON if owned
2. Filename includes snapshot date
3. `/api/snapshots/:id/rag_context` returns sanitized JSON (no account_numbers, etc.)
4. Non-owner: 403 for download
5. Non-admin: 403 for RAG endpoint
6. BI stub formats correctly (flat keys)
7. RAG includes user_id_hash with SHA256

## Test Cases
```ruby
describe "Api::SnapshotsController", type: :request do
  let(:user) { create(:user) }
  let(:other_user) { create(:user) }
  let(:snapshot) { create(:financial_snapshot, :with_full_data, user: user) }

  before { sign_in user }

  describe "GET /api/snapshots/:id/download" do
    it "downloads full JSON for owner" do
      get api_snapshot_download_path(snapshot)

      expect(response).to be_successful
      expect(response.content_type).to eq('application/json')
      expect(response.headers['Content-Disposition']).to include("financial-snapshot-")

      json = JSON.parse(response.body)
      expect(json['total_net_worth']).to eq(snapshot.data['total_net_worth'])
    end

    it "denies access to non-owner" do
      other_snapshot = create(:financial_snapshot, user: other_user)
      get api_snapshot_download_path(other_snapshot)

      expect(response).to have_http_status(:forbidden)
    end
  end

  describe "GET /api/snapshots/:id/rag_context" do
    let(:admin) { create(:user, role: :admin) }

    before { sign_in admin }

    it "returns sanitized JSON for admin" do
      get api_snapshot_rag_context_path(snapshot)

      expect(response).to be_successful
      json = JSON.parse(response.body)

      expect(json).to have_key('user_id_hash')
      expect(json).to have_key('exported_at')
      expect(json).not_to have_key('account_numbers')
      expect(json).not_to have_key('institution_ids')
    end

    it "denies access to non-admin" do
      sign_in user
      get api_snapshot_rag_context_path(snapshot)

      expect(response).to have_http_status(:forbidden)
    end

    it "includes SHA256 hash of user_id" do
      get api_snapshot_rag_context_path(snapshot)
      json = JSON.parse(response.body)

      expected_hash = Digest::SHA256.hexdigest("#{user.id}#{ENV['RAG_SALT']}")
      expect(json['user_id_hash']).to eq(expected_hash)
    end
  end

  describe "Tableau export stub" do
    it "returns flattened hash" do
      provider = Reporting::DataProvider.new(user)
      tableau = provider.to_tableau_json

      expect(tableau).to be_a(Hash)
      expect(tableau).to have_key('total_net_worth')
      expect(tableau).to have_key('allocation_equity')
      # No nested hashes
      tableau.values.each do |v|
        expect(v).not_to be_a(Hash) unless v.is_a?(Hash) && v.empty?
      end
    end
  end
end
```

## Workflow
1. Generate controller: `rails g controller api/snapshots download rag_context --skip-template-engine`

2. Add routes:
   ```ruby
   namespace :api do
     resources :snapshots, only: [] do
       member do
         get :download
         get :rag_context
       end
     end
   end
   ```

3. Implement `DataProvider#to_rag_context`:
   ```ruby
   def to_rag_context
     build_snapshot_hash.except(
       'account_numbers',
       'institution_ids',
       'raw_transaction_data'
     ).merge(
       user_id_hash: Digest::SHA256.hexdigest("#{user.id}#{ENV['RAG_SALT']}"),
       exported_at: Time.current,
       disclaimer: "Anonymized data for educational AI context"
     )
   end
   ```

4. Implement `DataProvider#to_tableau_json` (stub):
