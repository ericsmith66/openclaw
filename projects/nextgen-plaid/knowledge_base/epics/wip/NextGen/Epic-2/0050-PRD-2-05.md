# PRD-2-05: Snapshot – Holdings & Transactions Summary

## Overview
Extend JSON with top holdings summary and monthly transaction aggregates for Epic 3 holdings/transactions previews via DataProvider.

## Status
**Branch**: `feature/epic-2-financial-snapshots`
**Dependencies**: PRD-2-04 (Sector Weights)

## Requirements

### Functional
- **Top holdings**: Top 10 by `current_value` DESC
  - Structure: `[{ticker, name, value, pct_portfolio}, ...]`
  - `pct_portfolio`: value / total_positions_value
  - Fallback: Use raw Position data if enrichment missing

- **Monthly transactions** (last 30 days):
  - `income`: Sum positive amounts
  - `expenses`: Sum absolute value of negative amounts
  - `top_categories`: Top 5 by absolute amount `[{category, amount}, ...]`
  - Fallback: Use raw Transaction data

- Limit to prevent large JSON (top 10 holdings, top 5 categories)

### Non-Functional
- Return empty arrays when no data
- Summaries fit in reasonable JSON size

## Acceptance Criteria
1. Top holdings sorted descending by value, correct pct_portfolio
2. Transaction summary income/expenses match sums for last 30 days
3. Handles no data (empty arrays/zeros)
4. Fallback for missing names/categories works
5. Job runtime <5s

## Test Cases
```ruby
it "includes top holdings with percentages" do
  total = 0
  10.times do |i|
    value = (10 - i) * 10_000
    total += value
    create(:position, user: user, ticker: "TICK#{i}", name: "Company #{i}",
           current_value: value)
  end

  FinancialSnapshotJob.perform_now(user)
  snapshot = user.financial_snapshots.last

  holdings = snapshot.data['top_holdings']
  expect(holdings.size).to eq(10)
  expect(holdings.first['value']).to eq(100_000)
  expect(holdings.first['pct_portfolio']).to be_within(0.01).of(100_000.0 / total)
end

it "includes monthly transaction summary" do
  create(:transaction, user: user, amount: 5000, category: 'Salary', date: 15.days.ago)
  create(:transaction, user: user, amount: -2000, category: 'Rent', date: 10.days.ago)
  create(:transaction, user: user, amount: 1000, category: 'Old', date: 60.days.ago)

  FinancialSnapshotJob.perform_now(user)
  snapshot = user.financial_snapshots.last

  summary = snapshot.data['monthly_transaction_summary']
  expect(summary['income']).to eq(5_000)
  expect(summary['expenses']).to eq(2_000)
  expect(summary['top_categories'].size).to be <= 5
end
```
