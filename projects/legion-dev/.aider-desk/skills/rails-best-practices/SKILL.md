# Rails Best Practices Skill

This skill provides advanced expertise in developing and maintaining modern Ruby on Rails applications (version 8.0+).

## Core Expertise
- **Service Objects**: Designing decoupled, testable business logic in `app/services/`.
- **ViewComponents**: Building reusable UI components using the `view_component` gem.
- **RSpec Proficiency**: Writing clean, descriptive tests using FactoryBot, instance doubles, and shared examples.
- **Rails 8 Features**: Leveraging new Rails 8 features like Propshaft, modern solid gems, and improved performance APIs.
- **Database Optimization**: Advanced indexing, JSONB querying, and avoiding N+1 issues.

## Implementation Guidelines
- When asked to implement a new feature, prioritize creating a Service Object first.
- If UI is involved, suggest using a ViewComponent.
- Always include an RSpec plan with every implementation.
- Respect the `rails-base-rules.md` at all times.
